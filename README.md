# Randall Lowe – Portfolio Page

This portfolio page is a modular, JavaScript‑driven project designed to showcase my work, practice DOM manipulation, and demonstrate dynamic rendering using sessionStorage.

## Overview

This page was built using:

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Font Awesome (CDN)**

The project includes:

- A dynamic project renderer
- Accessibility features (dyslexic font, dark mode)
- A modal window
- Contact form feedback
- Session storage usage
- Modular file structure

## Dependencies

The following external resources are used:

- Font Awesome
    https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.0/css/all.min.css

No build tools or frameworks are required.
